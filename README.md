# CarGoCar - Ride Sharing Application

A simplified ride-sharing web application focused on coordinating carpool trips.

## Project Structure

- **CarGoCarAPI/**: ASP.NET Core REST API backend
- **CarGoCarWebApp/**: Blazor web application frontend
- **Documentation/**: Project requirements and design documents

## Getting Started

### Prerequisites

- .NET 8.0 SDK
- Visual Studio Code or Visual Studio

### Build and Run

```bash
# Build the solution
dotnet build

# Run API
dotnet run --project CarGoCarAPI

# Run Web App
dotnet run --project CarGoCarWebApp
```

## Features

- User registration and authentication
- Driver ride publishing
- Passenger seat reservations
- Admin moderation
- Role-based access control
